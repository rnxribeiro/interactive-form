# 📝 Formulário Interativo

Uma interface moderna de autenticação desenvolvida com HTML, SCSS e JavaScript, com animações suaves, background deslizante e design responsivo.

A aplicação conta com telas de login e cadastro integradas em uma única interface, proporcionando uma experiência fluida e agradável em diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da interface da aplicação.
* **SCSS:** Estilização, responsividade e implementação das animações e transições.
* **JavaScript:** Manipulação do DOM e controle da alternância entre as telas de login e cadastro.

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento Front-End, incluindo manipulação do DOM, responsividade, animações com CSS e criação de interfaces modernas e interativas.

O projeto foi construído como forma de aplicar na prática conhecimentos de desenvolvimento web moderno e aprimorar habilidades na construção de experiências visuais mais sofisticadas.

## ✨ Funcionalidades

* Alternância entre login e cadastro com JavaScript.
* Background deslizante com animações suaves.
* Transições sincronizadas entre formulários e painéis.
* Interface moderna inspirada em aplicações atuais.
* Layout totalmente responsivo para desktop e dispositivos móveis.
